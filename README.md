## How To RUn

1. Clone this repository
   ```
   $ git clone https://github.com/feriferdinan/chat-app.git
   ```
2. Install all depedencies on the package.json
   ```
      cd chat-app
      npm install
   ```
3. Run chat-app
   ```
   $ react-native run-android
   ```
