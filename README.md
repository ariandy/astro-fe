## How To Run

1. Clone this repository
   ```
   $ git clone https://github.com/ariandy/astro-fe.git
   ```
2. Install all depedencies on the package.json
   ```
      cd chat-app
      npm install
   ```
3. Run chat-app
   ```
   $ react-native run-android
   ```
